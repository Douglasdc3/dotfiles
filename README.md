# Dotfiles

## Manual Steps

### 1. Clone repo

```
git clone git@github.com:Douglasdc3/dotfiles.git .dotfiles
cd .dotfiles
```

### 2. Run install script

```
# Switch to ZSH shell
zsh
zsh install.sh
```

## 3. Miscellaneous 

* Install alfred workflows
* Setup CAPPS-LOCK -> CTRL mapping
* Setup Internet accounts
